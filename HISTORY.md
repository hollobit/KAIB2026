## 2026-03-07 (v3.1) - 유사성 분석 탭 개편

### 1. 탭 이름 변경
- "중복사업 분석" → **"유사성 분석"** 으로 메인 탭명 변경

### 2. 하위 탭 구조 도입
- 정책 클러스터와 동일한 pill 스타일 하위 탭 메뉴 추가 (최상단 배치)
- **유사성/중복성** (기본 탭): KPI 위젯 + BI 위젯 6종(히트맵, 스코어카드, 버블차트, 절감 시뮬레이터, 포트폴리오, 분류 대시보드) + 중복 의심 그룹 목록
- **키워드 유사성 검색**: 검색식 지원(AND/OR/NOT/구문) + 전체 필드 검색 + 검색 결과 간 유사도 분석(매트릭스/리스트) + 필터된 그룹 목록
- **유사 사업 탐색기**: 유사도 기준 슬라이더 + 탐색기 (콘텐츠 맞춤 높이)

### 3. 키워드 유사성 검색 기능
- **전체 필드 검색**: 사업명, 부처, 사업목적, 사업내용, 법적근거, 시행기관, 프로그램, 키워드, 내역사업, 사업관리자, AI도메인, 분야/섹터 등 모든 필드 대상
- **검색식 지원**: 기존 사업목록 탭의 `parseSearchExpr`/`matchSearchExpr` 재활용
  - 공백: AND, `|`: OR, `-`: 제외, `"구문"`: 정확한 구문, `()`: 그룹핑
- **검색 결과 간 유사도 분석**:
  - Jaccard(40%) + Cosine(60%) 복합 유사도 계산
  - 12개 이하: 매트릭스 테이블 (셀 클릭 시 비교 뷰)
  - 상위 30쌍: 리스트 형태 (유사도 바 + 부처/사업명 + J/C 세부 점수)
  - 색상 코딩: 60%↑ 빨강, 30%↑ 노랑, 그 이하 초록

### 4. KPI 위젯 compact 스타일

## 2026-03-07 (v3.2) - 유사성 검색 고도화

### 1. 검색 버튼 방식 전환
- 실시간 검색 → 검색 버튼 클릭 / Enter 입력 방식으로 변경

### 2. 검색 결과 유사도 분석 개선
- **진행률 표시**: 대량 쌍 계산 시 프로그레스 바 + 백분율 + 상세 단계 표시
- **Abort 토큰**: 새 검색 시 진행 중인 계산 자동 취소
- **쌍 보기 / 그룹 보기 토글**: Union-Find 기반 클러스터링 (유사도 30% 이상)
- **유사성 이유 표시**: 공통 토큰을 필드별로 분류 (사업명, 내역사업, 사업목적, 사업내용, 동일부처, 동일시행기관)
- **키워드 하이라이트**: 검색어를 결과 전체에서 `<mark>` 태그로 강조
- **필드별 매칭 정보**: 프로젝트별 어떤 필드에서 키워드가 매칭됐는지 표시
- **높이 고정 제거**: 유사도 분석 위젯의 고정 높이 제한 없음

### 3. 검색 대상 확장
- 20개 이상 필드 전체 검색: 사업명, 부처, 목적, 내용, 법적근거, 시행기관, 프로그램, 단위사업, 세부사업, 코드, 구분, 회계유형, 상태, 지원유형, 보조율, 키워드, 내역사업, 사업관리자, AI도메인, 분야/섹터

### 4. 검색어 기록
- localStorage 기반 검색 히스토리 (최대 20개)
- 클릭으로 이전 검색어 재활용, 개별/전체 삭제

### 5. 검색 결과 없음 표시
- 매칭 0건 시 안내 메시지 표시

### 6. 내역사업 레벨 유사성 검색
- "내역사업" 체크박스 추가
- 체크 시 프로젝트를 내역사업 단위로 분리하여 유사도 분석
- 각 내역사업은 부모 프로젝트 속성 상속 + 자체 이름/예산 보유
- 결과에 `← 상위사업명` 표시, 비교 클릭 시 부모 프로젝트 기준 상세 비교

### 7. 인쇄 및 Markdown 내려받기
- **인쇄하기**: 인쇄용 스타일의 새 창에서 현재 유사도 분석 결과 인쇄
- **Markdown 내려받기**: 검색 메타정보 + 상위 50쌍 테이블 + Union-Find 그룹별 사업 목록을 `.md` 파일로 다운로드

### 8. 내역사업 체크박스 즉시 반영
- 검색 결과가 있는 상태에서 내역사업 체크박스 변경 시 즉시 재검색

### 9. 매칭 1건 이하 시 내역사업 레벨 안내
- 사업 레벨 매칭 결과가 1건 이하일 때 내역사업 레벨 검색 건수 표시
- "내역사업 레벨 검색 결과는 체크박스를 체크하면 볼 수 있습니다" 안내 메시지

### 10. 검색 결과 헤더에 내역사업 레벨 건수 표시
- 사업 레벨 검색 시 헤더에 내역사업 건수 배지 표시, 클릭 시 내역사업 레벨로 전환

### 11. 유사도 알고리즘 선택
- 5종 알고리즘 드롭다운: Jaccard+Cosine (기본), Jaccard, Cosine, Dice (Sørensen), Overlap Coefficient
- 알고리즘 변경 시 검색어가 있으면 즉시 재검색
- 결과 헤더, 진행률 UI, 쌍 상세, 매트릭스 툴팁, Markdown 내보내기에 알고리즘 정보 반영
